Login page for project.

Goal was to learn how to implement Flask-Login library. I have created database where user name and password is stored. Usernames are not encripted, but passwords are. To secure passwords, passwords given through registration form are pushed through passlib library hash256 function with salting. Salting prevents storing two the same passwords for different users with the same value.

After logging in, in top right corner should appear greetings message with logout button.
All features with ability to change password through "Forgot password?" page are present.

Cookies for every user are stored for 10 minutes.
