Api Projects

This page has several Api projects on board. My goal was to prove I can both create API on my own, and use existing ones. Thats why I recycled my old webscraping project and implemented it on my website.

It works like this:

  • You can use form below to send Allegro url you want to scrap, or send POST request on /api/allegro:

    • {
    "base_url" : "Enter url to article you are interested in"
    "pages" : "Enter number of pages to scrap"
    }
  • Example POST:

    {
    "base_url" : "https://allegro.pl/listing?string=rower&bmatch=baseline-product-eyesa2-dict43-spo-1-4-0318"
    "pages" : "2"
    }
